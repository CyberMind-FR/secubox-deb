Touchless Control Methods

With the touchpad disabled, how should users interact with the Eye Remote?

You mentioned Bluetooth and WiFi as possibilities. The Pi Zero W has built-in WiFi and Bluetooth. Options:

1

Bluetooth Keyboard/Remote

Pair a mini Bluetooth keyboard or media remote
Arrow keys navigate, Enter selects, Escape goes back
Example: Rii mini keyboard, Fire TV remote

2

Bluetooth Phone App

Phone acts as touchpad/keyboard via Bluetooth HID
Apps like "Serverless Bluetooth Keyboard" or custom SecuBox app
Requires: Phone with BT, app install

3

Web Remote (WiFi)

Local web server on Pi Zero serves control page
Phone/laptop opens http://eye-remote.local:8080/control
Virtual buttons for navigation + settings
Requires: Same WiFi network or USB OTG

4

Display-Only (No Navigation)

Eye Remote is purely a status display
All control via flag files on USB storage
Mode changes require mount/unmount from host
Simplest, most reliable

5

Serial Console (ACM)

USB serial gadget already exists (ttyGS0)
Text menu over serial: screen /dev/ttyACM0 115200
Works: Always available via USB, no extra hardware

Select all that should be supported (multi-select enabled)
