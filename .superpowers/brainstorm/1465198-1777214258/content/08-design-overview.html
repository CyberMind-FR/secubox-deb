Eye Remote Swiss Army Dashboard — Design Overview

Complete design summary for approval

🎯 Core Concept

Transform Eye Remote from a simple metrics display into a Swiss Army multi-tool with 4 operating modes, touchless web-based control, and intelligent failover — all running on Pi Zero W with HyperPixel 2.1 Round (480x480).

🔄 Four Operating Modes

Mode Trigger Display
Dashboard SecuBox API responds (auto) 6 metric rings, real-time data
Local No API / fallback after 60s Pi Zero stats + icon grid
Flash /FORCE_FLASH flag file USB storage + progress + U-Boot
Gateway /FORCE_GATEWAY flag file Multi-SecuBox fleet view

🌐 Web Remote Control

Local web server at http://eye-remote.local:8080/control

Mode Switching WiFi Config Bluetooth Display Settings Device Manager System Actions SecuBox Control

⚡ Failover Behavior

  • API Loss: Stale data + "OFFLINE" badge → 15s gray fade → 60s Local mode
  • USB Replug: "Connecting..." spinner → green flash on success
  • Auto-reconnect: Background probe every 10s, silent recovery

🏗️ Architecture

Unified Python Agent — Single process extending existing code:

  • fb_dashboard.py → Extended with mode state machine
  • Embedded FastAPI server for Web Remote (:8080)
  • Async event loop handles display + web + SecuBox comms
  • System controls via subprocess (nmcli, bluetoothctl, etc.)
