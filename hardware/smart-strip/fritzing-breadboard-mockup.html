Mise en service côté Pi

Activer I²C : sudo raspi-config nonint do_i2c 0 puis détecter la Smart-Strip avec i2cdetect -y 1 (l'adresse 0x42 doit apparaître). Contrôle simple via Python avec smbus2 :

from smbus2 import SMBus
bus = SMBus(1)
bus.write_i2c_block_data(0x42, 0x10, [192, 78, 36]) # AUTH = orange charte
bus.write_i2c_block_data(0x42, 0x40, [0x55]) # heartbeat

Pilote complet (CDC + I²C unifié) dans host/secubox_smart_strip.py.
