# 🔐 Zero-Knowledge Proofs

ZKP Hamiltonian management

**Category:** Privacy

## Screenshot

![Zero-Knowledge Proofs](../../docs/screenshots/vm/zkp.png)

## Features

- Proof generation
- Verification
- Key management

## Installation

```bash
# Add SecuBox repository
curl -fsSL https://apt.secubox.in/install.sh | sudo bash

# Install package
sudo apt install secubox-zkp
```

## Configuration

Configuration file: `/etc/secubox/zkp.toml`

## API Endpoints

- `GET /api/v1/zkp/status` - Module status
- `GET /api/v1/zkp/health` - Health check
